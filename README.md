## Installation
### For windows users
(run those in command prompt/cmt/terminal)
For the robot to listen to our voice/speech
`pip install speechRecognition`

To speak out, or text to speech
`pip install pyttsx3`

control on browser
`pip install pywhatkit`

 wikipedia data
`pip install wikipedia`

 funny jokes
`pip install pyjokes`
